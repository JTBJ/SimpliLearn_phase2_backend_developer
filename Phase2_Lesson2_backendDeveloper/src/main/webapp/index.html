To run the project it may be necessary to create the databases I've used

The statement below creates the database. Copy and paste as needed.

Connection conn = DBConnection.getConnection(jdbcUrl, userName, password);

DBConnection.createManipulationStatement(conn, "CREATE DATABASE IF NOT EXISTS NewDatabase");

It may also be necessary to create the tables I've used

The statement below creates the tables. Copy and paste as needed.

DBConnection.createManipulationStatement(conn, "CREATE TABLE IF NOT EXISTS Student" +
    "(id INT not NULL AUTO_INCREMENT, " +
    "First_Name VARCHAR(255), "+
    "Last_Name VARCHAR(255), "+
    "Email VARCHAR(255), "+
    "PRIMARY KEY(id))");

Code to create the table for the Store Procedure

The table for the store procedure was created from the MySql command line. You can create a String of the below query statement and pass it to the createManipulationStatement() static method from the DBConnection class to create this table if needed.

CREATE TABLE eproduct(
 Id BIGINT primary key auto_increment,
 name VARCHAR(100),
 price decimal(10,2),
 date_added TIMESTAMP default now());

Code to create the Store Procedure

The store procedure was created from the MySql command line and is below.

CREATE PROCEDURE add_product(IN pname varchar(100), IN pprice decimal(10,2))
INSERT INTO eproduct (name, price) VALUES (pname, pprice);
